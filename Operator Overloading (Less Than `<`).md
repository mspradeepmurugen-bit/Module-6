## ğŸ’» Program
```
class A:
    def __init__(self,a):
        self.a=a
    def __lt__(self,o):
        if self.a < o.a:
            return "ob1 is less than ob2"
        else:
            return "ob2 is less than ob1"
ob1 = A(2)
ob2 = A(3)
print(ob1 < ob2)
```
